# Changelog

Все важные изменения в проекте будут документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.8.1] - 2026-04-09

### Добавлено
- Функция разблокировки достижений через Steam Network API
- Модуль `steamAchievementUnlocker.js` для работы с достижениями
- Функции `pauseGames()` и `resumeGames()` в farmManager для безопасной работы с достижениями
- Docker деплой на VDS
- Скрипт автоматического деплоя `deploy.ps1`
- Документация по деплою (DEPLOY.md, QUICKSTART.md)

### Исправлено
- Ошибка "Клиент Steam не найден" при разблокировке достижений
- Проблема с отключением клиента при попытке разблокировки
- Функция `checkSafety()` теперь правильно проверяет наличие клиента

### Изменено
- Обновлена библиотека `steamworks-ffi-node` для работы с достижениями
- Улучшена система управления фармом при работе с достижениями

## Процесс обновления

### Когда делаем обновление вместе:

1. **Я обновлю код** - исправлю баги, добавлю функции
2. **Обновлю версию** в package.json (например, 1.8.1 → 1.8.2)
3. **Добавлю запись** в этот CHANGELOG.md с описанием изменений
4. **Вы запустите деплой**:
   ```powershell
   cd C:\Users\Matychka\Desktop\steam-farm-bot
   .\deploy.ps1
   ```

### Типы версий:

- **1.8.1 → 1.8.2** (Patch) - исправление багов
- **1.8.2 → 1.9.0** (Minor) - новые функции
- **1.9.0 → 2.0.0** (Major) - большие изменения

### Проверка версии на VDS:

```bash
ssh root@81.17.154.12
cd /opt/steam-farm-bot
cat package.json | grep version
```
