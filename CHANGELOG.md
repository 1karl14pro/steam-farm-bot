# Changelog

Все важные изменения в проекте будут задокументированы в этом файле.

## [1.9.5.1] - 2026-04-13

### Исправлено
- 🔧 **Критическое**: Исправлен LogonSessionReplaced при восстановлении сессий
  - Увеличена задержка между запуском аккаунтов с 2 до 5 секунд
  - Теперь 10 аккаунтов восстанавливаются за ~50 секунд без конфликтов
- 🐛 Исправлена ошибка `db.getGamesWithHours is not a function`
  - Добавлена отсутствующая функция экспорта в database.js
- 🔄 Пересобраны все Docker контейнеры без кеша для применения всех исправлений

### Улучшено
- 📝 Обновлена документация в README
- 🔖 Синхронизированы версии в package.json и README

## [1.9.5] - 2026-04-12

### Добавлено
- ⏱️ **Точное отображение времени фарма** в формате ЧЧ:ММ:СС (например: 11:23:45)
- 📊 **Детальная статистика по играм**: показывается начальные → текущие часы (+нафармлено)
- 📈 **Отслеживание прогресса фарма**: для каждой игры видно, сколько часов нафармилось
- 🎯 **Прозрачность фарма**: точная информация о прогрессе каждой игры

### Изменено
- 🔄 Формат отображения времени изменен с "10ч" на "10:23:45"
- 📋 Улучшено отображение информации об аккаунтах в боте

## [1.9.4.1] - 2026-04-11

### Исправлено
- ✅ **Критическое**: Исправлен LogonSessionReplaced
  - Парсер больше не создает новые сессии при активном фарме
  - Устранено отключение аккаунтов каждые 6 часов
- ✅ Автоперезапуск сессий при отключении через 30 секунд
- ✅ Убран конфликт контейнеров
  - Отключен автоматический перезапуск каждые 24 часа

### Улучшено
- ⚡ Добавлены 4 новых индекса БД для ускорения запросов
- 🛡️ Добавлен Rate Limiting для защиты от банов Steam API
  - Использован Token Bucket алгоритм
- 🚀 Параллельное обновление кеша (по 3 аккаунта одновременно)
- 📊 Добавлен мониторинг использования памяти
