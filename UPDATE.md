# 🔄 Инструкция по обновлению бота

## Когда нужно обновление?

Обновление нужно когда:
- 🐛 Исправлены баги
- ✨ Добавлены новые функции
- 🔒 Обновления безопасности
- ⚡ Улучшения производительности

## Процесс обновления (вместе со мной)

### Шаг 1: Я делаю изменения
1. Исправляю код
2. Обновляю версию в `package.json`
3. Добавляю запись в `CHANGELOG.md`
4. Говорю вам: "Готово к деплою!"

### Шаг 2: Вы деплоите (1 команда!)
```powershell
cd C:\Users\Matychka\Desktop\steam-farm-bot
.\deploy.ps1
```

**Готово!** Новая версия автоматически:
- ✅ Загрузится на VDS
- ✅ Остановит старую версию
- ✅ Запустит новую версию
- ✅ Восстановит все аккаунты

### Шаг 3: Проверка (опционально)
```bash
ssh root@81.17.154.12
cd /opt/steam-farm-bot
```

## Автоматическое резервное копирование

Перед каждым обновлением скрипт `deploy.ps1` автоматически создает резервную копию на VDS.

## Проблемы при обновлении?

1. **Бот не запускается** - проверьте логи: `docker-compose logs`
2. **Аккаунты не подключаются** - проверьте .env файл
3. **Другие проблемы** - напишите мне, разберемся вместе!
