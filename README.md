# 🎮 Steam Farm Bot

Telegram бот для автоматического фарма часов в Steam играх.

## 📋 Описание

Steam Farm Bot - это Telegram бот для автоматического накручивания игрового времени в Steam. Бот работает круглосуточно, имитируя обычный запуск игр.

### ✨ Основные возможности

- 🎮 **Мульти-запуск**: Фарм до 30 игр одновременно на одном аккаунте
- 🔄 **24/7 работа**: Бот работает круглосуточно
- 👻 **Режим невидимки**: Возможность фармить в режиме "Невидимка"
- 💬 **Кастомный статус**: Установка собственного названия игры
- 📊 **Статистика**: Детальная статистика по нафармленным часам
- 🎯 **Цели фарма**: Установка целей и отслеживание прогресса
- ⏰ **Расписание**: Настройка расписания работы фарма
- 🔐 **Родительский контроль**: Поддержка PIN-кода для семейного просмотра
- 📚 **Библиотека игр**: Автоматическая загрузка библиотеки Steam
- 🏆 **Достижения**: Просмотр достижений игр
- 💼 **Автотрейды**: Автоматическое принятие трейдов-подарков
- 🔔 **Уведомления**: Уведомления о трейдах, заявках в друзья и событиях

## 🚀 Установка

### Требования

- Node.js 18+ 
- npm
- Telegram Bot Token (получить у [@BotFather](https://t.me/BotFather))
- Steam аккаунты с refresh токенами

### Локальная установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/yourusername/steam-farm-bot.git
cd steam-farm-bot
```

2. Установите зависимости:
```bash
npm install
```

3. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

4. Заполните `.env` файл своими данными:
```env
BOT_TOKEN=your_telegram_bot_token
ADMIN_IDS=your_telegram_id
BOT_USERNAME=YourBotUsername
PROXY_URL=http://username:password@host:port (опционально)
CRYPTO_API_TOKEN=your_crypto_bot_token (опционально)
FORCE_PROXY=false
```

5. Запустите бота:
```bash
npm start
```

### Docker установка

1. Соберите образ:
```bash
docker-compose build
```

2. Запустите контейнер:
```bash
docker-compose up -d
```

3. Просмотр логов:
```bash
docker-compose logs -f
```

## 🎯 Использование

### Команды бота

- `/start` - Запуск бота и главное меню
- `/menu` - Открыть главное меню

### Основные функции

1. **Добавление аккаунта**: Добавьте Steam аккаунт через QR-код или логин/пароль
2. **Выбор игр**: Выберите игры из библиотеки или добавьте по App ID
3. **Запуск фарма**: Запустите фарм для выбранных игр
4. **Мониторинг**: Отслеживайте статистику и прогресс в реальном времени
5. **Автотрейды**: Включите автоматическое принятие трейдов-подарков

### Получение refresh токена

Для работы бота необходим refresh токен Steam аккаунта. Бот поддерживает авторизацию через:
- QR-код (рекомендуется)
- Логин и пароль с Steam Guard кодом

## 🔧 Технологии

- **Node.js** - Серверная платформа
- **Telegraf** - Telegram Bot Framework
- **better-sqlite3** - SQLite база данных
- **steam-user** - Работа с Steam API
- **steam-session** - Авторизация в Steam
- **steam-tradeoffer-manager** - Управление трейдами
- **Docker** - Контейнеризация

## 📊 База данных

Проект использует SQLite для хранения данных:

- `users` - Пользователи бота
- `steam_accounts` - Steam аккаунты пользователей
- `games` - Игры для фарма
- `library_cache` - Кеш библиотек игр
- `farm_stats` - Статистика фарма
- `farm_goals` - Цели фарма
- `farm_schedules` - Расписания фарма
- `notifications` - Настройки уведомлений

## 🛡️ Безопасность

- Все токены хранятся в зашифрованном виде
- Данные не передаются третьим лицам
- Алгоритм имитирует обычный запуск игр
- Поддержка прокси для дополнительной безопасности
- Файлы .env и deploy.ps1 исключены из git
- Не храните конфиденциальные данные в коде
