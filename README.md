# 🎮 Steam Farm Bot

Telegram бот для автоматического фарма часов в Steam играх 24/7.

## 📋 Описание

Steam Farm Bot - это мощный инструмент для автоматического накручивания часов в Steam играх через Telegram. Бот работает на удаленных серверах 24/7, имитируя обычный запуск игр без риска получить VAC-бан.

### ✨ Основные возможности

- 🎮 **Мульти-запуск**: Фарм до 30 игр одновременно на одном аккаунте
- 🔄 **24/7 работа**: Бот работает круглосуточно на защищенных серверах
- 👻 **Режим невидимки**: Возможность фармить в режиме "Невидимка"
- 💬 **Кастомный статус**: Установка собственного названия игры
- 📊 **Статистика**: Детальная статистика по нафармленным часам
- 🎯 **Цели фарма**: Установка целей и отслеживание прогресса
- ⏰ **Расписание**: Настройка расписания работы фарма
- 🔐 **Родительский контроль**: Поддержка PIN-кода для семейного просмотра
- 📚 **Библиотека игр**: Автоматическая загрузка библиотеки Steam
- 💎 **Система подписок**: Триал и Premium подписки с разными лимитами
- 🎁 **Реферальная система**: Приглашайте друзей и получайте бонусы

## 🚀 Установка

### Требования

- Node.js 18+ 
- npm или yarn
- Telegram Bot Token (получить у [@BotFather](https://t.me/BotFather))
- Steam аккаунты с refresh токенами

### Шаги установки

1. Клонируйте репозиторий:
```bash
git clone https://github.com/yourusername/steam-farm-bot.git
cd steam-farm-bot
```

2. Установите зависимости:
```bash
npm install
```

3. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

4. Заполните `.env` файл:
```env
BOT_TOKEN=your_telegram_bot_token
ADMIN_IDS=123456789,987654321
BOT_USERNAME=YourBotUsername
PROXY_URL=http://proxy:port (опционально)
FORCE_PROXY=false
CRYPTO_API_TOKEN=your_crypto_pay_token (опционально)
STEAM_WEB_API_KEY=your_steam_api_key (опционально)
NODE_ENV=production
```

5. Инициализируйте базу данных:
```bash
npm run initdb
```

6. Запустите бота:
```bash
npm start
```

## 🎯 Использование

### Команды бота

- `/start` - Запуск бота и главное меню
- `/menu` - Открыть главное меню
- `/admin` - Админ-панель (только для администраторов)

### Основные функции

1. **Добавление аккаунта**: Добавьте Steam аккаунт с refresh токеном
2. **Выбор игр**: Выберите игры из библиотеки или добавьте по App ID
3. **Запуск фарма**: Запустите фарм для выбранных игр
4. **Мониторинг**: Отслеживайте статистику и прогресс в реальном времени

### Получение refresh токена

Для работы бота необходим refresh токен Steam аккаунта. Получить его можно через:
- Steam Desktop Authenticator (SDA)
- Извлечение из файла `.maFile`

## 💎 Система подписок

### Триал (7 дней бесплатно)
- До 5 аккаунтов
- До 4 игр на аккаунт
- Все основные функции

### Premium Базовый
- До 15 аккаунтов
- До 10 игр на аккаунт
- Приоритетная поддержка

### Premium Полный
- До 50 аккаунтов
- До 15 игр на аккаунт
- Все функции без ограничений

## 🔧 Технологии

- **Node.js** - Серверная платформа
- **Telegraf** - Telegram Bot Framework
- **better-sqlite3** - SQLite база данных
- **steam-user** - Работа с Steam API
- **steam-session** - Авторизация в Steam

## 📊 База данных

Проект использует SQLite для хранения данных:

- `users` - Пользователи бота
- `steam_accounts` - Steam аккаунты пользователей
- `games` - Игры для фарма
- `library_cache` - Кеш библиотек игр
- `farm_stats` - Статистика фарма
- `farm_goals` - Цели фарма
- `farm_schedules` - Расписания фарма
- `notifications` - Настройки уведомлений
- `pending_payments` - Ожидающие платежи

## 🛡️ Безопасность

- Все токены хранятся в зашифрованном виде
- Данные не передаются третьим лицам
- Владелец бота НЕ имеет доступа к вашему аккаунту Steam
- Алгоритм имитирует обычный запуск игр (безопасно для VAC)

## ⚠️ Отказ от ответственности

Вы используете бот на свой риск. Владелец бота не несет ответственности за возможные блокировки Steam аккаунтов. Фарм часов не нарушает правила Steam, но используйте с осторожностью.
